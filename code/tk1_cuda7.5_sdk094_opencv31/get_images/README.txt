#####    
#####   This sample demonstrates how to grab images and depth map with the ZED SDK using parallel threads.
#####
#####   =====================
#####   * Build the program *
#####   =====================
#####    
#####       Open a terminal in Grabbing thread directory and execute the following command:
#####           $ mkdir build
#####           $ cd build
#####           $ cmake ..
#####           $ make
#####    
#####   ===================
#####   * Run the program *
#####   ===================
#####    
#####       Open a terminal in build directory and execute the following command:
#####           $ ./ZED\ Optimized\ Grab [path to SVO file]
